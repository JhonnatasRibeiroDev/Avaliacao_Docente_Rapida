Explicação

Como você dever fazer no seu computador

Passo a passo:

1º passo, clique para copiar o código JavaScript abaixo.

const selects = document.querySelectorAll('select');
selects.forEach((select) => {
const secondOption = select.querySelectorAll('option')[1];
secondOption.setAttribute('selected', 'selected');
});

2º passo, dentro da página, clique com o botão esquerdo do mouse e vá em inspecionar.

3º passo, vá em console, cole o código e dê enter.

Pronto, agora só finalizar, e correr pro abraço.
